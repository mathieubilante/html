HTML5 Markup

HTML5 est la dernière version du langage HTML, utilisé pour créer des pages web. Il apporte de nouvelles balises sémantiques, un meilleur support multimédia et des formulaires enrichis. La structure de base d’une page HTML5 comprend : <!DOCTYPE html> pour déclarer HTML5, <html lang="fr"> comme conteneur principal, <head> pour les métadonnées (titre, encodage, viewport), et <body> pour tout le contenu visible. Les balises sémantiques importantes incluent : <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>, <figure>/<figcaption>, <video>, <audio>, <canvas>, et <form>. Les attributs courants sont : id, class, href, src, alt et target. Les bonnes pratiques consistent à utiliser les balises sémantiques, à ajouter des alt aux images, à organiser les fichiers dans des dossiers (/css, /js, /images) et à tester le site sur différents navigateurs et écrans.

CSS3 Styling

CSS (Cascading Style Sheets) est le langage utilisé pour mettre en forme le HTML. Il permet de définir les couleurs, polices, marges, tailles, alignements, positions, arrière-plans et animations des éléments sur une page web. CSS sépare le contenu HTML de sa présentation pour un site plus clair et maintenable.
